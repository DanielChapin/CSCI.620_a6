## Usage Instructions

Either set up Python virtual environment manually or run `make` and one will be generated for you in `.venv`.
If set up manually, install the dependencies in `requirements.txt`.

Run the main program by executing `python3 -m src.main` after activating the virtual environment.

## Command Line Interface

For command line argument details, run `python3 -m src.main --help`.
